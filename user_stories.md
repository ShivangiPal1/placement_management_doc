User Stories – AI-Powered Placement Management Platform

Student User Stories
•	As a student, I want to create an account so that I can access placement opportunities.
•	As a student, I want to complete my profile and upload my resume so that recruiters can view my qualifications.
•	As a student, I want to track the status of my applications so that I know whether I am shortlisted, selected, or rejected.
•	As a student, I want to receive notifications about new job openings and interview schedules.

Recruiter/Company User Stories
•	As a recruiter, I want to create a company profile so that students can learn about my organization.
•	As a recruiter, I want to post job opportunities so that eligible students can apply.
•	As a recruiter, I want to view student applications so that I can evaluate candidates.
•	As a recruiter, I want to shortlist candidates based on their profiles and resumes.
•	As a recruiter, I want to update application statuses so that students remain informed.

Placement Cell/Admin User Stories
•	As an admin, I want to manage student accounts so that student information remains accurate.
•	As an admin, I want to manage recruiter accounts so that only verified companies can access the platform.
•	As an admin, I want to monitor placement activities so that the placement process runs smoothly.
•	As an admin, I want to schedule and manage placement drives so that students and recruiters can participate efficiently.
