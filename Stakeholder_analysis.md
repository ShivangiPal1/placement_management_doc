# 🌟 Stakeholder Analysis
## AI-Powered Placement Management Platform  

---

##  1. Project Overview  

### Users Involved  
- **Students** – create profiles, upload resumes, and apply for jobs  
- **Recruiters/Companies** – post jobs and view applications  
- **Placement Cell/Admin** – manage students, recruiters, and placement activities  

---

###  Main Features  
- Login and profile management  
- Resume upload  
- Job posting and job applications  
- Candidate shortlisting  
- Application tracking  

---

###  Purpose  
The platform aims to make the placement process simple, organized, and easier for students, recruiters, and the placement cell.

---

##  2. Stakeholder Identification & Role Mapping  

###  Students (Primary Users)  
**Role:** Platform users  

**Responsibilities:**  
- Create and maintain profiles  
- Upload resumes  
- Apply for jobs  

**Goals:**  
- Access relevant job opportunities  
- Track application status  

**Pain Points:**  
- Difficulty tracking applications  
- Lack of clear guidance  
- Missing deadlines  

---

### Recruiters / Companies  
**Role:** Hiring partners  

**Responsibilities:**  
- Post job openings  
- Review applications  
- Shortlist candidates  

**Goals:**  
- Find suitable candidates quickly  
- Reduce hiring time  

**Pain Points:**  
- Too many irrelevant applications  
- Manual filtering effort  

---

###  Placement Cell / Admin  
**Role:** System manager  

**Responsibilities:**  
- Manage student data  
- Approve recruiters  
- Oversee placement activities  

**Goals:**  
- Ensure smooth placement process  
- Improve placement rates  
- Monitor progress easily  

**Pain Points:**  
- Manual tracking and coordination  
- Data mismanagement  

---

##  3. Power vs Interest Analysis  

| Stakeholder            | Power | Interest | Strategy          |
|------------------------|-------|----------|------------------|
| Placement Admin        | High  | High     | Manage Closely    |
| Recruiters             | High  | High     | Manage Closely    |
| Students               | Low   | High     | Keep Engaged      |

---

###  Engagement Strategy  

**Manage Closely (Admin & Recruiters):**  
- Provide dashboards for monitoring  
- Ensure system reliability  
- Collect regular feedback  

**Keep Engaged (Students):**  
- Maintain simple UI  
- Provide notifications and updates  
- Enable easy application tracking  

---

##  4. Stakeholder Needs vs Features  

| Stakeholder | Needs                     | Platform Features              |
|------------|--------------------------|-------------------------------|
| Students   | Easy application process | Profile + Apply system        |
| Students   | Track application status | Application tracking          |
| Recruiters | Filter candidates        | Shortlisting tools            |
| Recruiters | Post jobs easily         | Job posting                   |
| Admin      | Manage system            | Admin dashboard               |
| Admin      | Monitor progress         | Reports and tracking          |

---

##  5. Risks & Mitigation  

| Stakeholder | Risk                     | Mitigation                    |
|------------|--------------------------|-------------------------------|
| Students   | Low adoption             | Simple and user-friendly UI  |
| Recruiters | Irrelevant applications  | Better filtering system       |
| Admin      | System complexity        | Clear dashboard design        |

---

## 6. Communication Plan  

| Stakeholder | Communication Method | Frequency      |
|------------|---------------------|---------------|
| Students   | Notifications/Email | Regular        |
| Recruiters | Email updates       | Per job cycle  |
| Admin      | Dashboard access    | Real-time      |

---

## 💡 7. Key Insights  

- Placement Admins and Recruiters have the highest influence, so their needs directly impact system success.  
- Students require a simple and intuitive interface to ensure high engagement.  
- Application tracking is a core feature connecting all stakeholders and must be highly reliable.  

---

## Conclusion  

This stakeholder analysis highlights the importance of balancing administrative control, recruiter efficiency, and student usability to build a smooth and effective placement management platform.
